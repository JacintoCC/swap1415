*El concepto "persistencia" hace referencia a:*  
a) El servicio está disponible la mayor parte del tiempo.   
b) El balanceador redirige a todos los usuarios al mismo servidor.  
c) El balanceador mantiene a los usuarios en espera si el servicio no está disponible.  
d) El tráfico de un usuario se dirige al servidor que comenzó a atenderlo.  
Respuesta correcta: d)  

*¿Cómo podemos usar el DNS para balancear la carga y cuál es el principal inconveniente de hacerlo?*  
a) Asignar una misma dirección varias IP para repartir mediante round-robin. Round-robin es un algoritmo poco eficaz.  
b) Configurar el DNS para que conozca cuál es el servidor con menos carga y lo redirija a él. Es una configuración difícil y costosa.  
c) Asignar una misma dirección varias IP para repartir mediante round-robin. No sabemos si el servidor está disponible.  
d) Configurar el DNS para que conozca cuál es el servidor con menos carga y lo redirija a él. No sabemos si el servidor está disponible.  
Respuesta correcta: c)
